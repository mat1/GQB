How to create a query

To create a query you have to click on a node in the graph.

How to add a property filter

First add an node to the query. Then you can click the add property button (+ symbol) to add a property filter.

How to go back in a query

If you want select a pervious added node in the query, you have to use the breadcrumb navigation.

How to create a subquery

First you have to add an node to the query. After that you can start a subquery by a click on the subquery button.

Now you can add some nodes to the subquery. To finish the subquery press the stop subquery button.

How to create a branch

With branches you can define multiply paths to the same node. For example you can define a query with two differnt paths from the node User to the node Status:

  1. User to Place to Status
  2. User to Status

First you have to add an node to the query. After that you can start a branch by a click on the branch button.

Now you can add some nodes to the branch. To finish the branch press the stop branch button.

To add a second branch click the start query button again. Now you can add a differnt path to the node Status. It is necessary that your second branch ends on the same node as before. In this example the last node in the branch has to be the node Status.
